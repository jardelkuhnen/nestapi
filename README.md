## Installation

```bash
$ npm install
```

## Running the app

```bash
# Starting keycloak
 On keykloack directory start using a docker-compose file:
$ docker-compose up 

Start app:
# development
$ docker-compose up --build
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
